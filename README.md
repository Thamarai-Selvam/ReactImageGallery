# ReactImageGallery
Materialistic Image Gallery using Pixabay OpenSource API.

### pre-requisites
#### Base Reqs.
  Node.js  
  Pixabay API Key
 
  
#### Libraries/modules
##### run ```yarn install``` or ```npm install``` inside react-gallery dir. to install them  
  yarn/npm  
  sass  
  
    
  
## How to run
    cd react-gallery 
    yarn install
    yarn start
